🚀 Stellar Odyssey

Built to visualize the physics I once solved on paper — bringing velocity, gravity, and trajectories to life through code.

🌌 About

Stellar Odyssey is a beginner-friendly physics simulation project that transforms projectile motion equations into interactive visualizations.

The goal is simple: take concepts often studied in textbooks and bring them to life through programming.

Users can experiment with launch velocity, angles, and motion while observing how trajectories change in real time.

---

✨ Features

- Velocity Input
- Launch Angle Input
- Maximum Height Calculation
- Time of Flight Calculation
- Range Calculation
- 2D Trajectory Visualization
- Interactive Physics Learning

---

🧠 Physics Concepts

This project explores:

- Projectile Motion
- Kinematics
- Gravity
- Velocity Components
- Trajectory Analysis
